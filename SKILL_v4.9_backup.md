## 📋 标准化采集协议 (固化)

### 每助理必须执行
1. **≥3轮搜索**: 逐步扩大搜索范围
2. **多源验证**: 同一指标≥2个来源
3. **差异标注**: 差异>5%时明确标注
4. **时效性检查**: 优先最新数据
5. **置信度评级**: A/B/C/D

### 搜索轮次 (固化)
```
Round 1: 精准关键词，高置信源
Round 2: 补充验证，交叉核对 (R1不完整时)
Round 3: 仅当差异>5%时执行
```

## ✅ 中孚审核机制 (固化)

### 审核流程
```
10助理数据采集完成
    ↓
【中孚审核】 - 非自动审核
    ↓
审核通过 → 生成报告
审核不通过 → 返回助理重新采集
```

### 中孚审核职责
1. **逐条审核**: 每个数据点逐一核对
2. **来源验证**: 验证数据来源可靠性
3. **差异调查**: 差异>5%时调查原因
4. **置信度评级**: A/B/C/D评级
5. **质量评分**: 0-100分评分
6. **审核结论**: 通过/不通过/需补充

### 审核标准
| 评分 | 等级 | 结论 |
|------|------|------|
| 90-100 | A+ | 通过，优秀 |
| 80-89 | A | 通过，良好 |
| 70-79 | B | 通过，合格 |
| 60-69 | C | 需补充部分数据 |
| <60 | D | 不通过，重新采集 |

### 审核触发条件
- 10助理全部完成数据采集后
- 数据完整度<80%时
- 存在D级数据时
- 用户明确要求审核时

## ✅ 置信度评分标准 (固化)

| 等级 | 标准 | 使用条件 |
|------|------|----------|
| **A** | 官方API数据 | EIA API |
| **B** | 权威媒体/行业网站 | LNG168, 隆众, Mysteel |
