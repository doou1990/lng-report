# 纵横分析法融合方案

## 一、核心概念

**纵横分析法** = 横向扫描（广度）+ 纵向挖掘（深度）

### 横向维度（广度扫描）
- 跨市场对比（原油/LNG/煤炭/电力）
- 跨地域对比（亚洲/欧洲/北美）
- 跨时间对比（日/周/月/季/年）
- 多源数据交叉验证

### 纵向维度（深度挖掘）
- 历史周期回溯（3年/5年/10年）
- 因果链条拆解（价格→供需→库存→宏观）
- 趋势推演预测（短期/中期/长期）
- 关键转折点识别
