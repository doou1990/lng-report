## 1. 自然语言查询

### 功能说明

用户可以直接用中文提问，系统自动：
1. 识别查询意图
2. 生成SQL查询
3. 获取数据
4. 自动生成可视化
5. 生成文字分析

### 使用示例

```python
# 初始化查询引擎
from nl_query_engine import NLQueryEngine

engine = NLQueryEngine()

# 示例查询
queries = [
    "本周LNG价格趋势如何？",
    "比较Brent和WTI原油价格",
    "华东地区LNG价格",
    "最近7天JKM价格变化",
    "本月中国LNG市场概况"
]

# 执行查询
result = engine.query("本周LNG价格趋势如何？")

# 返回结果
{
    "query": "本周LNG价格趋势如何？",
    "intent": {
        "action": "trend",
        "data_type": "lng",
        "time_range": {"start": "2026-04-01", "end": "2026-04-06"}
    },
    "sql": "SELECT * FROM lng_prices WHERE date BETWEEN ...",
    "data": [...],  # 30条数据
    "visualization": {
        "type": "line",
        "title": "LNG价格趋势",
        "echarts_option": {...}
    },
    "analysis": "本周LNG价格呈现上涨趋势，JKM上涨2.3%...",
    "confidence": "A"
}
```

### 支持的查询类型

| 查询类型 | 示例 | 输出 |
|---------|------|------|
| **趋势查询** | "本周价格趋势" | 折线图 + 趋势分析 |
| **对比查询** | "比较Brent和WTI" | 柱状图/折线图 + 对比分析 |
| **区域查询** | "华东地区价格" | 表格/地图 + 区域分析 |
| **统计查询** | "本月平均价格" | 指标卡 + 统计分析 |
| **预测查询** | "下周价格预测" | 预测图 + 风险提示 |

## 2. 可视化增强

### 8种图表类型

| 图表类型 | 适用场景 | 自动选择条件 |
|---------|---------|-------------|
| **折线图** | 时间序列趋势 | 时间跨度>7天 |
| **柱状图** | 类别对比 | 对比分析 |
| **K线图** | 价格波动 | OHLC数据 |
| **面积图** | 累积趋势 | 展示总量变化 |
| **散点图** | 相关性分析 | 双变量关系 |
| **热力图** | 矩阵数据 | 多维度对比 |
| **仪表盘** | 关键指标 | 单一重要数值 |
| **组合图** | 多指标对比 | 价格+库存+利润 |

### 自动选择逻辑

```python
# 根据数据特征自动选择最佳图表
viz = LNGVisualizationLibrary()

# 自动选择
chart_type = viz.auto_select_chart(data, intent)

# 选择逻辑:
# - 时间序列 + 多维度 → 折线图/面积图
# - 对比分析 → 柱状图
# - 价格OHLC数据 → K线图
# - 双变量关系 → 散点图
# - 矩阵数据 → 热力图
# - 单一关键指标 → 仪表盘
# - 多指标组合 → 组合图
```

### 交互功能

所有图表支持：
- ✅ 数据缩放 (DataZoom)
- ✅ 工具栏 (Toolbox): 保存、刷新、数据视图
- ✅ 悬停提示 (Tooltip)
- ✅ 图例切换 (Legend)
- ✅ 区域缩放 (Brush)
- ✅ 响应式布局

## 3. MCP服务器设计

### 架构设计

```
LNG MCP Server
├── Tools (工具)
│   ├── get_crude_oil_price()     # 获取原油价格
│   ├── get_lng_price()           # 获取LNG价格
│   ├── get_china_lng_price()     # 获取中国LNG价格
│   ├── generate_lng_report()     # 生成报告
│   └── natural_language_query()  # 自然语言查询
├── Resources (资源)
│   ├── lng://prices/{date}       # 价格资源
│   ├── lng://reports/latest      # 最新报告
│   └── lng://history/...         # 历史数据
└── Prompts (提示模板)
    ├── price_analysis_prompt     # 价格分析
    └── market_summary_prompt     # 市场总结
```

### 使用方式

**Python客户端**:
```python
from mcp import ClientSession

async with ClientSession(read, write) as session:
    # 获取价格
    price = await session.call_tool("get_lng_price", {"date": "2026-04-06"})
    
    # 生成报告
    report = await session.call_tool("generate_lng_report", {"format": "html"})
    
    # 自然语言查询
    result = await session.call_tool(
        "natural_language_query", 
        {"query": "本周LNG价格趋势"}
    )
```

**Claude Code集成**:
```json
{
  "mcpServers": {
    "lng-market": {
      "command": "python",
      "args": ["lng_mcp_server.py"]
    }
  }
}
```

from nl_query_engine import NLQueryEngine

engine = NLQueryEngine()
