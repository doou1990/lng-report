## 三、融合机制详解

### 3.1 学习循环流程

```
每日 14:00 报告生成
         ↓
每日 14:05 自动触发学习循环
         ↓
┌─────────────────┐
│ Step 1: 评估    │
│ • 统计置信度分布 │
│ • 识别D级数据项  │
│ • 记录搜索日志   │
└─────────────────┘
         ↓
┌─────────────────┐
│ Step 2: 提取    │
│ • D级→数据源发现 │
│ • 失败→搜索优化  │
│ • 误判→审核进化  │
│ • 反馈→偏好学习  │
└─────────────────┘
         ↓
┌─────────────────┐
│ Step 3: 改进    │
│ • 小变更：自动   │
│ • 大变更：审批   │
│ • 记录变更日志   │
└─────────────────┘
         ↓
月度深化（每月第一个周日）
```

### 3.2 审批决策树

```
变更提案
    ↓
┌─────────────────────┐
│ 变更类型判断         │
├─────────────────────┤
│ • 关键词优化？      │ → 小变更 → 自动批准
│ • C级以上新源？     │ → 小变更 → 自动批准
│ • 阈值微调≤5%？     │ → 小变更 → 自动批准
│                     │
│ • C级以下新源？     │ → 大变更 → 待审批
│ • 阈值调整>5%？     │ → 大变更 → 待审批
│ • 新增必采数据项？  │ → 大变更 → 待审批
│ • 架构变更？        │ → 大变更 → 待审批
└─────────────────────┘
```

### 3.3 Fallback 机制

```
用户指令: "回退到 v6.2"
         ↓
┌─────────────────────────┐
│ 1. 暂停学习循环          │
│ 2. 切换至 v6.2-BASE SKILL│
│ 3. 保留 v6.3 配置        │
│ 4. 等待重新启用          │
└─────────────────────────┘
         ↓
用户指令: "启用 v6.3"
         ↓
┌─────────────────────────┐
│ 1. 恢复学习循环          │
│ 2. 加载 v6.3 配置        │
│ 3. 从上次状态继续        │
└─────────────────────────┘
```

## 五、安全保障机制

### 5.1 变更边界控制

| 层级 | 允许变更 | 禁止变更 |
|------|---------|---------|
| **执行层** | 无 | 数据采集代码、审核逻辑、输出格式 |
| **策略层** | 搜索关键词、数据源配置、审核阈值 | 执行流程、数据项定义 |
| **配置层** | 审批策略、触发时间、日志配置 | 版本锁定状态 |

### 5.2 审计追踪

```
logs/
├── learning_loop.log           # 学习循环执行日志
├── skill_updates.json          # 技能更新历史
data/
├── learning_loop_YYYY-MM-DD.json  # 每日学习结果
├── market_data_YYYY-MM-DD.json  # 每日市场数据
└── search_log.json             # 搜索日志
```

### 5.3 版本锁定保护

```
v6.2-BASE/SKILL.md  ← 🔒 完全锁定
         ↓
任何修改尝试 → 被拒绝 → 记录审计日志
         ↓
如需修改 → 用户明确审批 → 创建 v6.4 新版本
```

| 用户指令 | 系统响应 |
|---------|---------|
| "LNG报告" | 使用 v6.3 生成报告，自动触发学习循环 |

### 7.2 管理操作

| 用户指令 | 系统响应 |
|---------|---------|
| "回退到 v6.2" | 切换至基础版本，暂停学习循环 |
| "启用 v6.3" | 重新启用学习循环 |
| "审批变更" | 展示待审批的大变更列表 |
| "运行学习循环" | 手动触发学习循环 |

### 7.3 紧急操作

| 场景 | 用户指令 | 系统响应 |
|------|---------|---------|
| 报告异常 | "回退到 v6.2" | 立即切换至稳定版本 |
| 数据错误 | "暂停学习循环" | 停止自动优化，保持当前策略 |
| 撤销变更 | "撤销上次变更" | 回滚到上一版本配置 |
