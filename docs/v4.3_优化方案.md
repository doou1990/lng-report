### 3. 审核流程优化

#### v4.2 vs v4.3对比

```
v4.2流程 (易超时):
Layer 3: 12助理采集 → 完成
Layer 4: 中孚审核员(独立助理) → 重新搜索验证 → 120s超时

v4.3流程 (避免超时):
Layer 3: 12助理采集 → 完成
Layer 4: 
  - L1自动检查: 主代理执行(规则验证) → 10s
  - L2交叉验证: 主代理执行(多源对比) → 20s
  - L3异常检测: 主代理执行(统计检测) → 20s
  - L4人工审核: 仅当置信度<C时触发 → 按需
```

#### 分层审核实现
```python
# verification/layer_4_verification.py

def layer_4_verification(data: dict, agent_results: list) -> AuditReport:
    """第四层：多层验证 (v4.3优化版)"""
    
    # L1: 自动检查 (主代理执行，~10s)
    l1_result = l1_auto_check(data)
    if not l1_result.passed:
        return AuditReport(confidence='D', errors=l1_result.errors)
    
    # L2: 交叉验证 (主代理执行，~20s)
    l2_result = l2_cross_validation(agent_results)
    
    # L3: 异常检测 (主代理执行，~20s)
    l3_result = l3_anomaly_detection(data)
    if l3_result.is_anomaly and l3_result.severity == 'high':
        confidence = 'C'
    else:
        confidence = l2_result.confidence
    
    # L4: 人工审核 (仅当置信度<C时触发)
    if confidence in ['C', 'D']:
        return request_human_audit(data)
    
    return AuditReport(
        confidence=confidence,
        l1_passed=True,
        l2_result=l2_result,
        l3_result=l3_result
    )
```

#### 审核时间预算
| 层级 | 执行者 | 预计时间 | 条件 |
|------|--------|---------|------|
| L1 | 主代理 | 10s | 始终执行 |
| L2 | 主代理 | 20s | 始终执行 |
| L3 | 主代理 | 20s | 始终执行 |
| L4 | 人工/主代理 | 40s | 仅置信度<C |
| **总计** | - | **50-90s** | 大部分情况50s |
