# v6.1 混合子代理模式 - 架构说明

## 架构概览

```
┌─────────────────────────────────────────────────────────┐
│              主会话 (协调器)                              │
│         HybridSubAgentCoordinator                       │
└─────────────────────────────────────────────────────────┘
                           ↓
    ┌──────────────────────┼──────────────────────┐
    ↓                      ↓                      ↓
┌────────────┐      ┌────────────┐      ┌────────────┐
│  Phase 1   │      │  Phase 2   │      │  Phase 3   │
│ API子代理  │  →   │ 网页子代理  │  →   │ 审核子代理  │
│ (并行×3)   │      │  (串行×1)  │      │  (串行×1)  │
│ 30秒×3     │      │  300秒×1   │      │  60秒×1    │
└────────────┘      └────────────┘      └────────────┘
```

## 子代理分工

### Phase 1: API子代理 (并行)

| 子代理 | 数据域 | 采集内容 | 超时 | 数据等级 |
|--------|--------|----------|------|----------|
| 原油API | 国际原油 | Brent, WTI | 30秒 | A级 |
| LNG API | 国际LNG | JKM, TTF, HH | 30秒 | A级 |
| 库存API | 能源库存 | 美原油/天然气库存 | 30秒 | A级 |

**特点**: 3个子代理同时启动，互不影响，总耗时 ≈ 30秒

### Phase 2: 网页子代理 (串行)

| 子代理 | 数据域 | 采集内容 | 超时 | 数据等级 |
|--------|--------|----------|------|----------|
| 国内网页 | 中国LNG | 工厂价、浙江接收站、开工率 | 300秒 | B级 |

**特点**: 单独执行，不影响API子代理结果，失败不阻塞流程

### Phase 3: 审核子代理 (串行)

| 子代理 | 职责 | 输入 | 输出 | 超时 |
|--------|------|------|------|------|
| 中孚审核 | 质量评估 | 所有采集数据 | 评分报告 | 60秒 |

**特点**: 基于中孚经验系统化，四级评级+评分细则

## 执行流程

```
开始
  ↓
Phase 1: 启动3个API子代理 (并行)
  ├── 原油API → Brent, WTI
  ├── LNG API → JKM, TTF, HH
  └── 库存API → 美库存
  ↓ (等待全部完成，约30秒)
Phase 2: 启动网页子代理
  └── 国内网页 → 中国LNG价格
  ↓ (等待完成，约60-300秒)
Phase 3: 启动审核子代理
  └── 中孚审核 → 质量报告
  ↓
结束 (总耗时 < 6分钟)
```

## 超时配置

| 子代理类型 | 超时时间 | 说明 |
|------------|----------|------|
| API子代理 | 30秒 | API响应快，30秒足够 |
| 网页子代理 | 300秒 | 网页加载慢，需要5分钟 |
| 审核子代理 | 60秒 | 数据处理，1分钟足够 |

**总超时预算**: 30 + 300 + 60 = 390秒 (6.5分钟)

## 容错设计

```
API子代理失败 → 记录失败，继续执行
网页子代理失败 → 记录失败，国内数据标记为D级
审核子代理失败 → 记录失败，输出默认报告
```

**关键原则**: 单个子代理失败不阻塞整体流程

## 故障排查

### API子代理超时

```
症状: 原油API Agent timeout
解决: 检查API密钥，确认OilPriceAPI服务状态
```

### 网页子代理超时

```
症状: 国内网页 Agent timeout
解决: 
1. 检查网络连接
2. 确认Mysteel网站可访问
3. 考虑增加超时时间到600秒
```

### 审核子代理失败

```
症状: Evaluator failed
解决: 检查输入数据格式，确认JSON可解析
```
