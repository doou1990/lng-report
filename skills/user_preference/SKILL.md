## 学习机制

### 显式学习（用户明确指令）
```
用户: "以后把期货价差也加上"
    ↓
记录: user_preference.futures_spread = true
    ↓
下次报告自动包含期货价差
```

### 隐式学习（行为模式识别）
```
观察: 用户连续 5 次在 14:00 后询问报告
    ↓
推断: 用户关注 14:00 定时报告
    ↓
记录: user_preference.report_timing = "14:00_critical"
    ↓
14:00 未生成报告时主动提醒
```

### 反馈学习（用户纠正）
```
AI: "今日Brent价格 $85.5"
用户: "不对，应该是 $86.2"
    ↓
记录: 数据源 X 的 Brent 数据可能有偏差
    ↓
下次降低该源权重或标记待验证
```

## 偏好冲突解决

当新观察与已记录偏好冲突时：

| 冲突类型 | 解决策略 |
|----------|----------|
| 用户明确指令 vs 历史偏好 | 以最新指令为准 |
| 单次行为 vs 长期模式 | 需 3 次确认才更新 |
| 隐性推断 vs 用户否认 | 以用户否认为准 |
