# Claude Code 最佳实践

Claude Code最佳实践Skill，封装Agentic Loop、Subagent协作、Explore-Plan-Code工作流等核心方法论。

## 适用场景

- 复杂数据分析任务
- 多源数据采集与验证
- 高质量代码生成
- 需要可验证、可维护的交付物

## 核心原则

1. **Agentic Loop**: Gather → Act → Verify → Iterate
2. **Subagent协作**: 隔离长任务，并行处理
3. **Explore-Plan-Code**: 先探索，再规划，最后执行
4. **Verification-First**: 多源验证，标注置信度
5. **Context管理**: 及时清理，避免污染
