## 进化流程

### 触发条件
1. 用户反馈某数据被误判（正常数据标记为异常，或异常数据未检出）
2. 连续 3 天同一类型数据被标记异常（可能是规则过严）
3. 发现明显的异常数据未被检出（规则过松）

### 异常检测规则体系

#### 规则 1: 价格波动检测
**当前规则：**
```
IF |今日价格 - 昨日价格| / 昨日价格 > 15%:
    标记为异常
    置信度降级（如 B→C）
```

**进化方向：**
- 考虑地缘事件期间放宽阈值
- 增加连续波动检测

**进化记录：**
| 版本 | 阈值 | 变更原因 | 日期 |
|------|------|----------|------|
| v1.0 | 15% | 初始规则 | 2026-04-12 |
| v1.1 | 20% | 俄乌冲突期间频繁误判 | 待进化 |

#### 规则 2: 跨源差异检测
**当前规则：**
```
IF 同一指标多源差异 > 10%:
    标记为可疑
    取中位数，置信度降级
```

**进化方向：**
- 区分正常差异（不同统计口径）vs 异常差异
- 建立各指标的正常差异区间

**各指标正常差异区间（待学习）：**
| 指标 | 正常差异区间 | 说明 |
|------|-------------|------|
| Brent/WTI | ±2% | 基准原油，差异小 |
| JKM/TTF | ±5% | 区域市场，差异中等 |
| 国内LNG | ±8% | 区域分散，差异较大 |
| 库存数据 | ±10% | 统计口径不同 |

#### 规则 3: 时效性检测
**当前规则：**
```
IF 数据日期 < 今日 - 3天:
    标记为过时
    置信度降级
```

**进化方向：**
- 不同数据类型设置不同阈值
- 周末/节假日特殊处理

**各指标时效性阈值：**
| 指标类型 | 当前阈值 | 建议阈值 | 说明 |
|----------|----------|----------|------|
| 现货价格 | 3天 | 1天 | 价格变动快 |
| 期货价格 | 3天 | 1天 | 实时交易 |
| 库存数据 | 3天 | 7天 | 周度发布 |
| 进口数据 | 3天 | 30天 | 月度统计 |

#### 规则 4: 趋势异常检测（待添加）
**提案规则：**
```
IF 连续 3 天同向变动 > 10%:
    标记为趋势异常
    触发深度分析
```

**适用场景：**
- 价格连续上涨/下跌
- 库存连续累积/去化

## 误判案例库

### 案例 1: 俄乌冲突期间价格误判
- **时间**: 2026-04-XX
- **数据**: JKM 价格单日上涨 18%
- **规则反应**: 标记为异常，降级为 C 级
- **实际情况**: 地缘冲突导致，属正常市场反应
- **改进**: 阈值 15% → 20%，增加地缘事件检测

### 案例 2: 周末数据延迟误判
- **时间**: 2026-04-XX
- **数据**: 周一使用周五库存数据
- **规则反应**: 标记为过时（>3天）
- **实际情况**: 周末无数据更新，属正常
- **改进**: 周末阈值延长至 4 天

## 变更审批流程

```
规则变更提案
    ↓
影响评估（会影响哪些数据项？）
    ↓
[微调 ≤5%] → 自动批准 → 生效
    ↓
[大调整 >5%] → 用户审批 → 生效/拒绝
```
