## 🎓 中孚校核能力提升训练

### 1. 数据质量审核标准训练
**审核维度**：
1. **真实性审核**：是否为真实数据，禁止估算数据
2. **准确性审核**：数据是否准确，误差范围是否合理
3. **完整性审核**：所需数据是否齐全，缺失数据是否说明
4. **时效性审核**：数据是否及时，更新时间是否合理
5. **一致性审核**：不同来源数据是否一致，矛盾是否解释
6. **相关性审核**：数据是否与分析主题相关，冗余是否剔除

**审核评分体系**：
- **优秀 (90-100分)**：所有数据真实准确，来源透明，时效性好
- **良好 (75-89分)**：主要数据真实准确，少量数据需要改进
- **合格 (60-74分)**：基本数据真实，但需要较多改进
- **不合格 (<60分)**：数据质量不达标，需要重新执行

### 2. 数据来源验证训练
**验证方法**：
1. **直接验证**：访问数据来源网站验证数据
2. **交叉验证**：使用多个数据源验证同一数据
3. **逻辑验证**：基于行业知识和逻辑判断数据合理性
4. **历史验证**：与历史数据对比验证趋势合理性

**验证工具**：
```bash
# 使用exa验证数据来源
mcporter call 'exa.crawling_exa(urls: ["https://example-data-source.com"], maxCharacters: 2000)'

# 使用web_fetch验证API数据
web_fetch https://api.example.com/data

# 使用scrapling抓取验证
# ./run.sh fetch "https://example.com" "table.data-table"
```

### 3. 异常数据识别训练
**异常类型**：
1. **数值异常**：超出合理范围的数据
2. **趋势异常**：与历史趋势明显不符的数据
3. **来源异常**：来自非权威或可疑来源的数据
4. **时间异常**：更新时间异常或时效性差的数据
5. **一致性异常**：不同来源数据矛盾的数据

**识别方法**：
- **统计方法**：均值、标准差、异常值检测
- **逻辑方法**：基于行业知识的合理性判断
- **对比方法**：与历史数据、同行数据对比
- **验证方法**：重新获取数据验证

### 4. 审核报告编写训练
**报告结构**：
1. **审核概况**：审核时间、审核对象、总体评分
2. **分项审核**：各助理数据质量详细审核
3. **问题分析**：发现的主要问题和原因分析
4. **改进建议**：具体的改进措施和建议
5. **审核结论**：总体审核结论和后续要求

**报告质量要求**：
- 问题描述具体明确，有数据支持
- 改进建议可操作，有时限要求
- 评分客观公正，有评分依据
- 结论清晰明确，有后续行动计划
